Project Overview

SNACKS (Sensor Network Around Charlottetown's Key Surroundings) is a project that aims to give Charlottetown residents an easy way to participate in citizen science, and is inspired heavily by a similar project in the Netherlands.

The backbone of this project is LoRaWAN technology, a wireless communication standard that allows for long distance transmission of small packets of data. In particular, we are using The Things Network to move data from Arduino devices to our database. We have also utilized Amazon Web Services (AWS) services for various parts of the system. All of our code is open source and available on GitHub, licensed under the GNU GPL.

Future Direction

There are a number of sensor types that can be integrated with our system. At the moment, we have prototype devices running on AA batteries that gather temperature and humidity data once every hour. However, this system is not limited to temperature and humidity sensing. Since it only relies on LoRaWAN communication, any sensor that can be connected to an Arduino device could be integrated into this system with minimal effort. For example, there exist cheap Arduino sensors that can measure light levels, there are several well-documented Arduino projects online that measure rainfall, and there are also sensors available that sense air quality. These sensors all vary in cost and usefulness, but they can all be added into this system and report data concurrently.

One potential use case for temperature measurement in a system like this is to identify "micro-climates" within a city. It can perhaps be misleading to assume that the temperature as measured at the Charlottetown airport is the same temperature as the whole city. Could areas near a pond be cooler or warmer at different times of the year? Does construction of a parking lot or major building affect the temperature of a previously forested area? What is the long term effect of construction projects on localized temperature?

Hopefully, the most interesting insights from data gathered in this way will come from questions that cannot be articulated right now. The benefit of a citizen science project is that in the future, when there is a large amount of data available to inspect, an interested citizen may come up with their own question that reveals some unexpected insight about the City of Charlottetown.
