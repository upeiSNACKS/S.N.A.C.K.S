Project Overview

SNACKS (Sensor Network Around Charlottetown's Key Surroundings) is a project that aims to give Charlottetown residents an easy way to participate in citizen science.

The backbone of this project is LoRaWAN technology, a wireless communication standard that allows for long distance transmission of small packets of data. In particular, we are using The Things Network to move data from our Arduino devices to our database.

This project is inspired heavily by a similar project in the Netherlands

Future Applications

There are a number of sensors that could be integrated with our system. At the moment, we have prototype devices running on AA batteries that gather temperature and humidity data once every hour.
